PartTimer Web Interface Design

Innovative Employment Startup

PartTimer is an online service aiming to revolutionise the online employment process. Through PartTimer, employers approach candidates; job-seekers only need to set up their profile, and employers can easily search through applicants using requirements such as location and availability to help find the perfect employee.

Our team designed and developed the web interface for PartTimer. After collecting all the user stories for the project and deciding on an MVP, two distinct flows were created for the seperate roles. User testing sessions were held with wireframe prototypes to ensure usability of the interface.
