Domain Name Commission Website Redesign

Creating the .nz Web Experience

Refreshing and improving the web presence of New Zealand's Domain Name Commission Limited (DNCL).

The site was given a clean, professional design while enhancing the responsive viewing experience on all devices. The updated colour scheme and imagery better improved the alignment to their company culture.

The DNCL holds a vast archive of policies, news, disputes and important documents, so it was a goal from the beginning to ensure the website is easy to navigate despite its large content base.

Through several carefully executed information architecture sessions, we were able to sort the contents into five main sections, clear second-level navigation groups, and added multiple avenues to access the most important pages.​​​​​​​
