NZ Parliament Messaging Interface

Welcoming you to our Parliament

Digital signage solution for the New Zealand Parliament, displaying important information such as Select Committee room schedules and availability, welcome messages, location way-finding, tour schedules, and more.

We designed and developed an informational interface to be displayed on the screens. The system needed to be compatible with the existing Parliamentary Service Calendar System​​​​​​​ to populate bookings.

The interface was carefully designed to ensure the visual style matched their existing digital presence. Meeting strict compatibility and accessibility guidelines was a key objective of this project.
